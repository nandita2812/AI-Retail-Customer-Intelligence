# AI-Retail-Customer-Intelligence
AI-powered system for customer segmentation, churn prediction, and product recommendation

🧠 AI-Powered Retail Customer Intelligence System

📌 Overview

This project is an end-to-end data science solution designed to analyze retail customer behavior, identify high-value customers, predict churn, and recommend products using machine learning techniques.

---

🚀 Key Features

- 🔹 Customer Segmentation using RFM Analysis and K-Means Clustering
- 🔹 Churn Prediction using Logistic Regression
- 🔹 Product Recommendation System using Apriori Algorithm
- 🔹 Interactive Dashboard using Streamlit
- 🔹 Data Visualization for business insights

---

📊 Dashboard Insights

- Customer segment distribution
- Churn analysis
- Top customers by spending
- Product recommendations
- Interactive product search

---

🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit

▶️ How to Run

1. Install required libraries:
   pip install pandas numpy scikit-learn streamlit matplotlib

2. Run the dashboard:
   streamlit run app.py
